## 删除build
```powershell
Remove-Item -Recurse -Force build
```
---

## 编译命令
```powershell
cmake -S . -B build `
  -DCMAKE_TOOLCHAIN_FILE="D:\tools\vcpkg\scripts\buildsystems\vcpkg.cmake" `
  -DVCPKG_TARGET_TRIPLET=x64-windows `
  -DWITH_FFMPEG=OFF `
  -DWITH_CHANNEL_URBDRC=OFF `
  -DWITH_URBDRC=OFF `
  -DWITH_SDL=OFF `
  -DWITH_SERVER=OFF `
  -DWITH_SHADOW=OFF `
  -DWITH_PROXY=OFF `
  -DWITH_CLI=ON `
  -DWITH_CLIENT=ON `
  -DWITH_SAMPLE=OFF
```
